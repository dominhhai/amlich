# Installation
Global scope:
`npm install licham -g`

Local scope:
`npm install licham --save`

# Usage
## Command Line
CLI options

    -h, --help          output usage information
    -V, --version       output the version number
    -d, --date [date]   show calendar for specific date (ddMMyyyy|MMyyyy|yyyy)
    -y, --year          show year calendar
